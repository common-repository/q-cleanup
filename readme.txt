This plugin allows you to delete unused and leftover files from upload dir.
In one click you can rid of all unwanted files and reduce space usage.



== Description ==

Some very useful plugins like Regenerate Thumbnails leaves old files behind, so it is good practice to clean after it.


Features:
*Remove all unused files from upload dir (Unattached files to posts or pages),
*Remove all leftover files (forgotten files in upload dir).



== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `q-cleanup.zip` to the `/wp-content/plugins/` directory
2. Unzip archive.
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Navigate to 'Settings' -> 'Q Cleanup' menu
5. Clean up!
